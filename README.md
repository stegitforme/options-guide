# Options Strategies — A Practical Guide

A clean, interactive guide to options strategies for someone transitioning from futures trading.

## Sections

- **Strategy Rankings** — All major options strategies ranked, tabbed by tier
- **The Wheel Strategy** — Recommended starting point, with vetted video resources
- **Momentum Options** — Deep ITM call buying system with full position sizing math
- **Long-Term ETFs** — SPMO vs VOO vs QQQ comparison
- **GPIQ** — Covered call ETF as a sideways-market income tool
- **Recommendations** — Suggested sequence for actually approaching this

## Stack

Pure HTML, CSS, and vanilla JS. No build step, no framework. Hosted on GitHub Pages.

## Local Preview

Open `index.html` in a browser. That's it.
